# FitCoach Bot - بوت المدرب الذكي لكمال الأجسام

## 1. الرؤية والمفهوم

**FitCoach** هو مدربك الشخصي الرقمي الذي يعمل على مدار الساعة. يجمع بين العلم الحديث والخبرة العملية ليقودك في رحلة بناء الكتلة العضلية بطريقة منظمة وفعالة.

**الشخصية**: مدرب حازم لكن داعم، يتحدث بأسلوب عربي واضح ومباشر، يشبه المدرب المحترف في الصالة الرياضية.

**الرسالة الأساسية**: "أنا هنا لأ教练ك. اتبع البرنامج، سترى النتائج."

## 4. الميزات والتفاعلات

### 4.1 إدارة الملف الشخصي
| الحقل | النوع | التحقق |
|-------|-------|--------|
| العمر | 14-70 | أرقام فقط |
| الطول | 140-220 سم | أرقام فقط |
| الوزن | 30-200 كجم | أرقام فقط |
| نسبة الدهون | 3-50% | اختياري |
| الخبرة | مبتدئ/متوسط/متقدم | قائمة |
| أيام التمرين | 1-7 | أرقام |
| الهدف | ضخامة/تنشيف/بناء | قائمة |

**التفاعل**: wizard متعدد الخطوات مع التحقق الفوري

### 4.2 التتبع اليومي
```
🍳 الإفطار ← [إضافة وجبة] → [السعرات] [البروتين] [الكارب] [الدهون]
🥗 الغداء ← [إضافة وجبة] → [السعرات] [البروتين] [الكارب] [الدهون]
🍽️ العشاء ← [إضافة وجبة] → [السعرات] [البروتين] [الكارب] [الدهون]
🍼 وجبات خفيفة ← [إضافة وجبة] → [السعرات] [البروتين] [الكارب] [الدهون]
💧 الماء ← [أكواب] (250ml لكل كوب)
😴 النوم ← [ساعات]
📊 ملخص اليوم
```

### 4.3 التخطيط الغذائي الذكي
**حساب الاحتياج اليومي**:
- BMR = 10 × الوزن + 6.25 × الطول - 5 × العمر + 5 (ذكور)
- TDEE = BMR × معامل النشاط
- معامل النشاط:
  - خامل: 1.2
  - نشاط خفيف: 1.375
  - نشاط متوسط: 1.55
  - نشط جداً: 1.725
  - رياضي محترف: 1.9

**توزيع المغذيات حسب الهدف**:
| الهدف | بروتين | كارب | دهون |
|-------|--------|------|------|
| ضخامة | 2 غ/كغ | 55% | 25% |
| تنشيف | 2.5 غ/كغ | 30% | 25% |
| بناء | 2.2 غ/كغ | 40% | 30% |

### 4.4 نظام التدريب
**أقسام الجسم الرئيسية**:
- صدر
- ظهر
- أكتاف
- ذراعيك ( biceps + triceps)
- ساقيك
- بطن

**أنواع التمرين**:
- تمارين حرة (Free weights)
- تمارين الآلة (Machine)
- تمارين الجسم (Bodyweight)
- تمارين الكابل (Cable)

**كل تمرين يحتوي**:
- اسم التمرين
- العضلة المستهدفة
- عدد المجموعات
- التكرارات
- الراحة بين المجموعات
- صورة/فيديو توضيحي (رابط)
- بدائل

### 4.5 تحليل الأداء
- مقارنة الوزن الأسبوعية (رسمة بيانية)
- معدل التغير
- تقدير نسبة الدهون
- تنبيهات الذروة والانسحاب
- اقتراحات للتعديل

### 4.6 الذكاء التكيفي
```
الخوارزمية:
1. تحليل البيانات خلال 4 أسابيع
2. إذا وزن ثابت +Calories > TDEE → زيادة البروتين
3. إذا وزن ثابت + البروتين كافي → تعديل التمرين
4. إذا وزن ينزل بسرعة → تحذير + زيادة السعرات
5. إذا تقدم ممتاز → زيادة تدريجية
```

### 4.7 التقارير
- تقرير يومي مفصل
- تقرير أسبوعي مع رسمة بيانية
- تقرير شهري شامل
- مقارنة الفترات الزمنية

### نموذج قاعدة البيانات
```sql
-- جدول المستخدمين
CREATE TABLE users (
    user_id INTEGER PRIMARY KEY,
    telegram_id BIGINT UNIQUE,
    name TEXT,
    age INTEGER,
    height REAL,
    weight REAL,
    body_fat REAL,
    experience_level TEXT,
    workout_days INTEGER,
    goal TEXT,
    activity_level TEXT,
    created_at DATETIME,
    updated_at DATETIME
);

-- جدول التقدم اليومي
CREATE TABLE daily_progress (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    date DATE,
    weight REAL,
    calories INTEGER,
    protein REAL,
    carbs REAL,
    fats REAL,
    water REAL,
    sleep_hours REAL,
    notes TEXT,
    FOREIGN KEY (user_id) REFERENCES users(user_id)
);

-- جدول الوجبات
CREATE TABLE meals (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    date DATE,
    meal_type TEXT,
    name TEXT,
    calories INTEGER,
    protein REAL,
    carbs REAL,
    fats REAL,
    FOREIGN KEY (user_id) REFERENCES users(user_id)
);

-- جدول التمارين
CREATE TABLE workouts (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    date DATE,
    exercise_id INTEGER,
    sets INTEGER,
    reps INTEGER,
    weight REAL,
    notes TEXT,
    FOREIGN KEY (user_id) REFERENCES users(user_id),
    FOREIGN KEY (exercise_id) REFERENCES exercises(id)
);

-- جدول التمارين المرجعي
CREATE TABLE exercises (
    id INTEGER PRIMARY KEY,
    name_ar TEXT,
    name_en TEXT,
    muscle_group TEXT,
    equipment TEXT,
    difficulty TEXT,
    description TEXT,
    video_url TEXT
);

-- جدول الإعدادات
CREATE TABLE settings (
    user_id INTEGER PRIMARY KEY,
    language TEXT DEFAULT 'ar',
    notifications_enabled BOOLEAN DEFAULT 1,
    reminder_time TEXT,
    theme TEXT DEFAULT 'dark',
    FOREIGN KEY (user_id) REFERENCES users(user_id)
);
```

## 7. المصادر العلمية

### المراجع المعتمدة
1. **ACSM (American College of Sports Medicine)**
   - إرشادات التمارين المقاومة
   - توصيات اللياقة القلبية

2. **ISSN (International Society of Sports Nutrition)**
   - بروتوكولات المكملات
   - توقيت التغذية

3. **PubMed Articles**
   - بحث البروتين والكتلة العضلية
   - تأثير النوم على التعافي
   - أهمية прогрессияالتحميل

4. **مبدأ الازدواجية** (Principle of Double Progression)
5. **مبدأ التحميل التدريجي** (Progressive Overload)
6. **مبدأ التخصيص** (Specificity Principle)
